# sankaku jogi man
🔺 三角じょうぎマン vs ぶんどキング 角度バトル！ 📐

> **「角度なんて難しくてわからない？」**
> **そんなお悩みは、ダンボールをまとったじょうぎ星の王子「三角じょうぎマン」が解決する！**

本プロジェクトは、全国の悩める小学4年生に向けて、YouTube展開を想定して企画された教育用キャラクター「三角じょうぎマン」を主役とした、完全無料・インストール不要のインタラクティブ角度学習ゲーム（Webアプリケーション）です。

## 📖 ストーリー ＆ キャラクター紹介

### 🔺 三角じょうぎマン（じょうぎ星の王子・ジョウギル）

ダンボールで作った手作りの三角定規を身にまとった、エネルギッシュでちょっとコメディチックなヒーロー。

* **右手の武器**：「30度のじょうぎのけん（剣）」（30°-60°-90°）
* **左手の盾**：「45度のじょうぎのたて（盾）」（45°-45°-90°）
* **特徴**：バンダナの三角形は王家のロイヤルマーク。口を開けると三角の歯がきらりと光り、目にはやる気の炎が灯る！

### 👑 ぶんどキング（ぶんどき帝国・皇帝）

頭が黄色い分度器になっている、いつもニヤニヤ笑っている悪役。しかしその正体は、行方不明になった我が子（三角じょうぎマン）の成長を見守り、地球の子供たちに角度を学ばせるために、わざと悪役を演じていた「じょうぎ星の王様（オヤジ）」だった…！？

## 🎮 ゲームの主な特徴

小学4年生の算数「角の大きさ」の単元に完全対応した3つのステージで構成されています。

### 1. 小学4年生の教科書に準拠した3ステップ学習

* **第1戦：めもりのナゾ（基礎知識）**
  * 「1目盛りは1度であること」「一般的な分度器は180度まではかれること」をクイズ形式で楽しく学びます。
* **第2戦：角度を読みとれ！（実践・分度器の読み取り）**
  * 画面上の分度器から、示された角度をピタリと当てるクイズです。
  * **動的ランダム出題**：遊ぶたびに出題される角度（30度、45度、60度、90度、120度、135度、150度など）がランダムに変化するため、繰り返しドリル学習が可能です。
  * **左がわ/右がわ基準の完全サポート**：4年生が一番つまずきやすい「左の0から測るか、右の0から測るか」の違いを、分かりやすいビジュアルと解説でサポートします。
* **第3戦：角度の名前マスター（特別な角度）**
  * 90度（直角）、180度（平角）、360度（1回転）など、角度の特別な呼び方や回転の概念をマスターします。

### 2. 子どもが夢中になる「ピコピコ動く」ゲーム演出

* **レトロゲーム風サウンド（Web Audio API）**
  * 外部音源ファイルを一切使わず、ブラウザの音声合成機能だけで奏でる懐かしい 8bit 風のオリジナル BGM、正解時のファンファーレ、アタック効果音を内蔵。
* **楽しいアクションアニメーション**
  * クイズに正解すると、三角じょうぎマンがジャンプして敵に斬撃アタック！ぶんどキングがピョコピョコとダメージを受け、HP ゲージがリアルタイムで減少します。
* **感動の親子再会マルチエンディング**
  * クイズを全問クリアすると、ぶんどキングのHPが0になり、あの衝撃の親子設定が明かされる感動（？）のエンディングが待っています。

### 3. 計算し尽くされたバリアフリー＆ビジュアル設計

* **正確ではみ出さない分度器 Canvas**
  * JavaScriptのCanvas描画を最適化し、測る向きに合わせて半透明のガイド円弧が美しく表示されます。
* **目盛りの数字が重ならない親切設計**
  * 外側の数値（左から測定用）と内側の数値（右から測定用）の描画半径を大きく離し、数字が絶対に重ならないように整理。目の疲れないフォントサイズを採用しています。
* **小学4年生向け語彙の最適化**
  * 難しい漢字を排除、または適切なフリガナ付きのひらがな表記に変更し、小学生が一人でも迷わず遊べるテキストに仕上げています。

## 🛠 技術スタック ＆ 動作環境

外部のサーバーや重いフレームワークは一切不要。このリポジトリの `.html` ファイルを1つダウンロードするだけで、オフラインでもスマートフォンでも一瞬で起動します。

* **フロントエンド**：HTML5, JavaScript (ES6+), Canvas API
* **スタイリング**：Tailwind CSS (CDN経由)
* **サウンドシステム**：Web Audio API (純粋なコード演奏システム、音源ファイル不要)
* **グラフィック**：インラインSVGによる超軽量・高解像度キャラクター描画
